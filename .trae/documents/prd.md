## 1. 产品概述

高炉铁水脱硫过程可视化大屏——面向重型冶金工业的实时监控与数据分析平台。系统通过处理来自现场数万个传感器的吹氩、喷粉历史作业数据，在高炉纵切面图上以散点粒子喷射效果动态呈现脱硫粉剂浓度分布，并在侧边面板同步绘制硫元素含量下降速率的微积分曲线，为工艺工程师提供全流程、高精度的脱硫过程洞察。

- 目标用户：冶金工艺工程师、生产调度员、设备运维人员
- 核心价值：将数十 GB 级异构传感器数据转化为直观、同步、可交互的可视化画面，提升脱硫工艺优化效率

## 2. 核心功能

### 2.1 用户角色

| 角色 | 核心权限 |
|------|----------|
| 工艺工程师 | 查看全量数据、回放历史脱硫过程、调整时间窗口 |
| 生产调度员 | 实时监控当前炉次脱硫进度、查看关键指标 |
| 设备运维人员 | 监控传感器状态、异常报警 |

### 2.2 功能模块

1. **主监控大屏**：高炉纵切面可视化 + 侧边同步曲线面板 + 顶部状态栏
2. **数据回放控制**：时间轴播放/暂停/倍速、时间窗口选择

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主监控大屏 | 顶部状态栏 | 当前炉次号、脱硫阶段、运行时长、关键指标实时数值 |
| 主监控大屏 | 高炉纵切面可视化 | 基于Deck.gl的WebGL粒子喷射效果，动态展示脱硫粉剂浓度分布 |
| 主监控大屏 | 侧边硫含量曲线面板 | ECharts绘制硫含量(ppm)下降曲线及其微积分（下降速率）曲线，与主视图时间轴严格同步 |
| 主监控大屏 | 时间回放控制条 | 底部时间轴，支持播放/暂停/倍速/拖拽定位 |
| 主监控大屏 | 传感器数据面板 | 料位、温度、流量等关键传感器实时数值仪表盘 |

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深空黑(#0a0e17) + 熔岩橙(#ff6b35) + 钢蓝(#00d4ff) + 硫磺黄(#ffd700)
- **按钮风格**：扁平化、微圆角、发光边框效果，工业风
- **字体**：Rajdhani（工业数据展示字体）+ Source Sans 3（正文），数字使用等宽风格
- **布局风格**：全屏大屏布局，左侧主视觉区（高炉纵切面），右侧数据面板，顶部状态栏，底部时间控制条
- **图标风格**：线性图标，发光描边，科技感

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 主监控大屏 | 顶部状态栏 | 深色背景，发光文字，炉次号/阶段/时长/关键指标 |
| 主监控大屏 | 高炉纵切面 | 全屏WebGL Canvas，高炉轮廓线框图，橙色粒子喷射动画，浓度热力叠加层 |
| 主监控大屏 | 侧边曲线面板 | 深色半透明面板，双Y轴曲线图（硫含量+下降速率），同步时间标记线 |
| 主监控大屏 | 时间控制条 | 底部横条，播放/暂停按钮，倍速选择，可拖拽时间指针 |
| 主监控大屏 | 传感器面板 | 圆形仪表盘+数字，料位/温度/流量三组 |

### 4.3 响应式

桌面端优先，适配1920×1080及以上分辨率大屏显示。最小支持1366×768。

### 4.4 3D/粒子场景指引

- 环境：深色工业风背景，微弱网格底纹
- 粒子效果：从喷枪位置向外喷射的橙色粒子流，浓度越高粒子越密、越亮
- 交互：鼠标悬停显示局部浓度数值
- 后处理：辉光效果(bloom)，粒子拖尾
